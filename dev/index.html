ConstraintTrees.jl – Tidy constraints for linear programming

Build statusDocumentation

ConstraintTree provides a simple data structure ConstraintTree for organizing the contents of constraint-based linear problems. As a main goal, it abstracts over the distinction between constraints and variables, allowing much tidier representation for many kinds of complex constraint systems. The primary purpose is to support constraint-based metabolic modeling within COBREXA.jl.
